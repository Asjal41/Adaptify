How It Works

Three pillars of intelligent, personalized education

🎯

IQ Assessment

Students complete a cognitive assessment measuring logical reasoning, working memory, pattern recognition, and problem-solving to build a personalized profile.

📚

RAG Knowledge Base

Teachers upload course materials (DOCX, PDF, TXT) that get chunked and stored in ChromaDB. The AI retrieves relevant context for every assignment.

🤖

AI-Generated Assignments

Groq's Llama 3.3 70B generates game-based, zone-structured assignments adapted to each student's cognitive strengths and weaknesses using CLT principles.
